# PersonaOn — Agents Discovery Survey (Question Bank)

**Date:** 2026-06-09
**Purpose:** Learn what each audience (ICP) expects from PersonaOn — especially from the upcoming **Agents** feature (agents that carry out small-to-medium tasks for the user). Findings will shape the **onboarding flow, the questions we ask during onboarding, and the product roadmap.**

**How to read this doc:** Each question lists a **type** so the survey app knows how to render it:
`single` (one choice) · `multi` (many choices) · `scale` (1–5 / slider) · `text` (free response) · `email`.
Questions marked **[branch]** appear only for the matching role selected in Q1. Everything else is asked of everyone.

> Keep it short-feeling: aim for 4–6 minutes. Most questions optional except Q1.

## Section D — Agents (the core of this survey)  *(everyone)*

**D1. Which tasks would you happily delegate to an agent?** `multi`
